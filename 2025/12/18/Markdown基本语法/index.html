Markdown 基本语法指南

上次更新: 2026-02-25 22:28:20

Markdown 是一种轻量级标记语言，用于格式化纯文本，使其易于转换为HTML。以下是常用语法及编辑器操作提示（以 Typora 等常见编辑器为例）：

编辑器操作：

  • 快捷键：Ctrl + 1 到 Ctrl + 6 可以快速将当前行设置为对应的1-6级标题。
  • 操作：行首输入 # 号后跟一个空格，再输入标题内容。

文本样式

语法：

编辑器操作：

  • 粗体：选中文字，按 Ctrl + B。
  • 斜体：选中文字，按 Ctrl + I。
  • 删除线：选中文字，按 Alt + Shift + 5（部分编辑器适用）。

列表

无序列表

语法：

编辑器操作：

  • 新建：行首输入 -、* 或 + 后跟一个空格，然后输入内容，按 Enter 会自动创建下一个列表项。
  • 结束：连续按两次 Enter 或按 Backspace 删除自动生成的列表符号即可退出列表。

有序列表

语法：

编辑器操作：

  • 新建：行首输入 1. 后跟一个空格，然后输入内容。编辑器会自动编号后续项目。
  • 结束：连续按两次 Enter 或按 Backspace 删除自动生成的编号即可退出列表。

链接和图片

语法：

编辑器操作：

  • 链接：选中文字，按 Ctrl + K，然后输入网址。
  • 图片：直接拖拽图片到编辑器（需编辑器支持），或按上述语法手动输入。

引用

语法：

编辑器操作：

  • 新建：行首输入 > 后跟一个空格。
  • 结束：连续按两次 Enter 即可退出引用块。

代码

行内代码

语法：

编辑器操作：

  • 选中文字，按 `Ctrl + Shift + ``（部分编辑器适用），或在代码前后输入反引号 ```。

代码块

语法：

 ```python def hello(): print("Hello, World!") ```

或使用缩进（四个空格或一个制表符）：

编辑器操作：

  • 输入 ````` 后跟语言名称（如 ````python），然后按 Enter` 即可创建一个代码块。在代码块内，Tab键用于缩进，而不是跳到下一个输入框。
  • 在支持代码块的编辑器中，代码通常会有语法高亮。

表格

语法：

编辑器操作：

  • 在 Typora 等编辑器中，可以直接右键插入表格。
  • 也可以手动输入表头和分割线（第二行的 ---|---），编辑器会自动识别并生成表格。按 Tab 键可以在单元格间切换。

水平线

语法：

编辑器操作：

  • 输入 --- 或 *** 后直接按 Enter 即可生成一条水平线。

常用技巧

  1. 换行：在行尾添加两个空格后按 Enter，或直接空一行（产生新段落）。
  2. 强制换行：使用 <br> HTML标签。
  3. HTML支持：可以在Markdown中直接使用HTML标签来实现更复杂的格式，如 <center>居中</center> 或 <font color="red">红色文字</font>。

扩展语法（部分编辑器支持）

Markdown 的简洁性使其成为文档编写、笔记记录和网页内容创作的理想选择。不同的平台（如 GitHub、Typora、VS Code）可能会有自己的扩展语法，建议在使用前了解具体的平台特性。
